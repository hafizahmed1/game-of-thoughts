# game-of-thoughts

This project investigates whether Large Language Models (LLMs) can understand structured rule systems and generate new games with logically consistent rules.

## Research Question
Can large language models understand existing game rules and generate new games with logically consistent rules?

## Project Overview
Games provide structured environments with clear rules, constraints, and objectives. This project evaluates how well LLMs can interpret such rule systems and apply them in two tasks:

1. **Game Understanding**
   - The model is given the rules of an existing game (e.g., Tic-Tac-Toe).
   - It must explain the rules, detect inconsistencies, and suggest valid moves.

2. **Game Generation**
   - The model is asked to create a new game.
   - The generated rules are analyzed for completeness and logical consistency.

The goal is to study whether language models can maintain structured reasoning when dealing with rule-based systems.
